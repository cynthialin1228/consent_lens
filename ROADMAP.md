# ConsentLens Roadmap

This file keeps the current MVP stable while outlining the next improvements in a clear order.

## Baseline

The current version is the original working MVP and should remain the stable reference point.

## Version 1.1

Goal: make flagged terms faster to review on long pages.

- Add `Find next` and `Find previous` navigation
- Add current match counter like `2 / 14`
- Auto-scroll the selected term into view
- Add stronger active highlight styling for the selected term
- Add keyboard shortcuts for next and previous navigation

## Version 1.2

Goal: make the extension feel easier and more human-friendly.

- Add a floating on-page navigation bar
- Add a compact results summary on the page
- Add filter chips for `Privacy`, `Money`, `Tracking`, and `Legal rights`
- Add `High risk only` quick filter
- Add better first-run guidance in the popup

## Version 1.3

Goal: improve usefulness and trust.

- Expand the local legal-risk dictionary
- Reduce false positives with context rules
- Detect likely consent regions before scanning the full page
- Improve explanation wording to be even more plain-language
- Let users add custom flagged terms

## Version 2.0

Goal: expand beyond standard webpages.

- Add PDF support for text-based PDFs
- Add page-aware navigation for flagged PDF terms
- Add side panel results list with clickable jump links
- Add export of flagged terms from the current page

## Suggested GitHub Issues

- Preserve MVP baseline
- Add next and previous navigation
- Add on-page counter and active term focus
- Add floating navigator
- Improve dictionary coverage and context handling
- Add PDF support phase 1
