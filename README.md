# ConsentLens

ConsentLens is a Chrome extension that highlights important legal and consent-related terms on webpages using local, in-browser analysis only.

## Current MVP

- Highlights risky or important terms on normal webpages
- Shows hover explanations in plain language
- Includes simple highlight presets
- Rescans dynamic page content
- Stores settings locally in Chrome

## Load In Chrome

1. Open `chrome://extensions`
2. Turn on Developer Mode
3. Click `Load unpacked`
4. Select this project folder
5. Open a normal webpage and refresh it once after reloading the extension

## Current Limitations

- Webpages first, PDF support later
- Local dictionary-based detection only
- No server or AI processing
- Best results on privacy policies, terms pages, cookie banners, and consent screens
